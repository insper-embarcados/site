# Módulos

Neste primeiro laboratório expert, estarão disponíveis **três módulos**, dos quais cada aluno deverá escolher um:

* **IA**: Introduz a aplicação de inteligência artificial embarcada para detecção de padrões de movimento utilizando dados da IMU (aceleração).
* **RTOS**: Aprofunda o uso de RTOS, com foco em medição e análise do comportamento do sistema para garantir execução correta. Também aborda o uso de SMP (Symmetric Multiprocessing), possibilitando execução multicore.
* **Bluetooth**: Apresenta o uso de hardware Bluetooth para comunicação entre o microcontrolador (uC) e o computador, utilizando esse protocolo.
